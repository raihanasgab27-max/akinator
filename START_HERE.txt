📊 WHAT'S HAPPENING BEHIND THE SCENES:
═════════════════════════════════════════════════════════════════════════

   When you run generate_db.py:

   1. Script starts
   2. Asks you for URL (lokal/Colab)
   3. Tests connection to Ollama
   4. Asks for model name (llama2, mistral, neural-chat)
   5. Asks for confirmation
   6. Sends prompt to Ollama: "Generate Akinator database..."
   7. Ollama AI processes for 2-5 minutes
   8. Returns JSON with database structure
   9. Script validates the JSON
   10. Script counts items & nodes
   11. Script saves to akinator_db.json
   12. Script shows success message! ✅

   Database structure:
   - Binary decision tree
   - 30-45 Indonesian-focused items
   - Points to distinguishing questions
   - Leaf nodes contain guesses
   - Completely automatic! 🤖


🎮 SAAT BERMAIN (Game Loop):
═════════════════════════════════════════════════════════════════════════

   1. Script loads akinator_db.json
   2. Starts at root node (first question)
   3. Asks question
   4. You answer YES → traverse "yes" branch
   5. You answer NO → traverse "no" branch
   6. Repeat until reach leaf node
   7. Leaf node has AI's guess
   8. AI asks: "Correct?"
   
   If YES:
      → AI wins! ✅
      → Stats updated
      → Play again
   
   If NO:
      → Stats: wrong_guesses++
      → Asks: "What were you thinking?"
      → Asks: "Distinguishing question?"
      → Updates tree
      → Saves updated database
      → Next time AI is smarter! 🧠
